HTML Cheat Sheet



Tables

Tag Name Element Description
Table <table> The top-level Table tag, used to define the HTML Table
Table Head <thead> The Table Head tag is used to define the "head" row in an HTML Table
Body <tbody> The table body tag is used to define the main section of the HTML Table
Table Row <tr> The table row tag is used to define the individual rows of the HTML Table
Table Data <td> The table data tag is nested within a table row tag to define the actual data contained within a row of the HTML Table. Table data tags can used the colspan attribute to span across columns by supplying a numeric value representing the number of columns to span.
Table Footer <tfoot> The Table footer tag is used to define the "footer" row in an HTML Table
